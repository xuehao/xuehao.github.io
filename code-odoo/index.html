Odoo 开发

Odoo 提供了一个快速应用开发框架，非常适合创建商业应用。这类应用通常用于保留业务记录，增删改查操作。Odoo 不仅简化了这类应用的创建，还提供了看板、日历、图表等视图的丰富组件，用于创建好看的用户界面。

本专栏记录学习 Odoo 的点点滴滴，了解企业应用开发流程，以及软件在企业管理中的应用。

技术栈包括：

  • HTML/CSS/JavaScript（Bootstrap/SCSS）
  • Python（MVC）
  • PostgreSQL
