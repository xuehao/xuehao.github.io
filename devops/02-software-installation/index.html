2.1 添加用户到“sudoers”

在装完系统后，CentOS 默认普通用户是无法使用“sudo”命令的，这时我们就需要将登陆的用户加入“/etc/sudoers”文件中。

  • 切换到超级用户，打开“/etc/sudoers”文件：

      $ su -
  $ vi /etc/sudoers

  • 修改文件内容：

    找到下面这行，并在下面插入新行：

      root    ALL=(ALL)       ALL

    内容是：

      xxx    ALL=(ALL)       ALL   # xxx是你的用户名

    这个文件是只读的，保存时需要加“!”，强制保存。

2.3 通过 Software Collection 管理软件版本

Software Collection 可以方便的管理软件的不同版本，我们通过安装git来说明。

# 1. 安装 Software Collection：
$ sudo yum install centos-release-scl

# 2. 搜索软件包：
$ yum search git

# 3. 安装软件包：
$ sudo yum install rh-git29

# 3.1 软件包安装后，可以查看安装情况，该命令会列出安装过的所有软件包：
$ scl --list

# 4. 启用软件包：
$ scl enable rh-git29 bash

注意，这些软件包在重启后不会激活。Software Collection 设计初衷就是在不影响原有配置的前提下，让新旧软件能一起运行。不过你可以通过~/.bashrc加载 Software Collection 提供的“enable”脚本来实现自动启用。

source /opt/rh/rh-git29/enable

2.4 通过源码编译安装

有时会遇到一些软件，没有提供官方或第三方软件包，所以只能通过源代码手动编译安装。手动编译安装，看起来比较恐怖，但仔细阅读官方源码提供的安装说明，也不是什么难事。编译安装经常会遇到依赖库缺失的问题，以下我们通过编译安装 Emacs 来说明。

  • 下载 Emacs 源码：

    Emacs 等软件在系统软件仓库里，并没有最新版本，所以为了使用类似 Prelude、Spacemacs 这样的第三方配置文件，我们需要安装最新版的软件。

    Emacs 最新版本目前是 Emacs 26

  • 解压 Emacs 源码：

      $ tar -zxvf emacs26.1.tar.gz

  • 生成 Makefile 配置文件：

      $ ./configure --prefix=/opt/emacs26.1  # --prefix 指定安装路径

    ./configure 会检测平台是否包含编译需要的所有依赖，生成 Makefile 配置文件为编译做准备。

    在 CentOS 7 中，执行完出现了以下几个错误：

    • 没有安装 GTK+：
        ...
  configure: error: No X toolkit could be found.
  ...
  $ yum search gtk3 # 安装之前，先搜索软件包
  ...

  gtk3.x86_64 : GTK+ graphical user interface library
  gtk3-devel.i686 : Development files for GTK+
  gtk3-devel.x86_64 : Development files for GTK+
  ...
  $ sudo yum install gtk3-devel # 一般选择安装带有 -devel 后缀的软件

      libXpm libjpeg libpng ...

  • 编译并验证：

      $ make # 编译
  $ src/emacs -Q # 执行程序

    make 命令将源代码进行编译生成可执行程序，编译后的可执行程序在 “/src” 目录下。编译一切正常的话，已经可以直接运行程序了，Emacs 提供了第二个命令来验证程序。

  • 安装：

      $ sudo make install

  • 加入系统路径：

    安装完之后，记得加入系统路径，将以下内容添加到 “./bashrc” 中：

      PATH=/opt/emacs26.1/bin:$PATH
  export PATH

    这样可以直接在命令行中打开 Emacs：

      $ source .bashrc
  $ emacs --version
  GNU Emacs 26.1

以上就是通过源码编译安装一个软件的过程。
