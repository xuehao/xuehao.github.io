macOS 制作 Linux USB 启动盘

发表于 2018-12-26

  • 1. 转换系统镜像格式
  • 2. 查看U盘编号
  • 3. 写入前先卸载U盘
  • 4. 将系统写入U盘

有一个老的笔记本，运行 Windows 比较吃力，所以测试一些 Linux 发行版，目的是找一个驱动管理好，运行流畅的系统。以下记录一下 macOS 下制作 USB 启动盘的步骤：

1. 转换系统镜像格式

3. 写入前先卸载U盘

~/Downloads                                                                    
▶ diskutil umountDisk /dev/disk3
Unmount of all volumes on disk3 was successful

4. 将系统写入U盘

~/Downloads                                                                    
▶ sudo dd bs=1m if=linuxmint-19.1-cinnamon-64bit.dmg of=/dev/rdisk3
Password:
