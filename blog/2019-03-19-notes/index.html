安装 cslib 的步骤和问题

发表于 2019-03-19

  • 1. 解压文件，此命令会生成一个cslib文件夹
  • 2. 切换到 cslib 文件夹
  • 3. 使用make构建
  • 4. 将gccx文件添加到PATH路径
  • 5. 测试安装

1. 解压文件，此命令会生成一个cslib文件夹

 $ sh cslib.shar

2. 切换到 cslib 文件夹

3. 使用make构建

注意ubuntu系统需要安装csh工具以及x11开发包

$ sudo apt install csh x11-dev
$ make

4. 将gccx文件添加到PATH路径

PATH=~/cslib:$PATH
export PATH

5. 测试安装

$ gccx -o house house.c 
$ ./house
