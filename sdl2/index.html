SDL 学习笔记

学习完一本C语言入门书后，就可以尝试使用第三方库了。尽快减少玩具代码的练习，学习库和接口的相关技术，接触大型项目磨练技能。

SDL 就是一套很好玩的开源跨平台多媒体开发库，使用C语言写成，在类 Unix 平台上仅依赖C运行时库(glibc) 。

简单的测试程序

以下两个小程序可以测试安装是否正确。第一个程序是命令行界面，输出初始化信息是否正常。第二个程序是图形界面，显示一个空白窗口，2000ms后会自动关闭。

  • 命令行界面：

      // test_sdl.c
  #include <SDL2/SDL.h>

  int main(int argc, char *argv[])
  {
      if ((SDL_Init(SDL_INIT_VIDEO | SDL_INIT_AUDIO) == -1))
      {
          printf("Could not initialize SDL2: %s.\n", SDL_GetError());
      }
      else
      {
          printf("SDL initialize OK!\n");
      }

      SDL_Quit();

      return 0;
  }

  • 图形界面：

      // hello_sdl.c
  #include <SDL2/SDL.h>
  #include <stdio.h>

  const int SCREEN_WIDTH = 640;
  const int SCREEN_HEIGHT = 480;

  int main(int argc, char *args[])
  {
      SDL_Window *window = NULL;
      SDL_Surface *screenSurface = NULL;

      if (SDL_Init(SDL_INIT_VIDEO) < 0)
      {
          printf("SDL could not initialize! SDL_Error: %s\n", SDL_GetError());
      }
      else
      {
          window = SDL_CreateWindow("Hello SDL2", SDL_WINDOWPOS_UNDEFINED,
                                  SDL_WINDOWPOS_UNDEFINED, SCREEN_WIDTH,
                                  SCREEN_HEIGHT, SDL_WINDOW_SHOWN);

          if (window == NULL)
          {
              printf("Window could not be created! SDL_Error: %s\n", SDL_GetError());
          }
          else
          {
              screenSurface = SDL_GetWindowSurface(window);
              SDL_FillRect(screenSurface, NULL,
                          SDL_MapRGB(screenSurface->format, 0xFF, 0xFF, 0xFF));
              SDL_UpdateWindowSurface(window);
              SDL_Delay(2000);
          }
      }

      SDL_DestroyWindow(window);
      SDL_Quit();

      return 0;
  }
