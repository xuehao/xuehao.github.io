2.1 添加用户到“sudoers”

在装完系统后，CentOS 默认普通用户是无法使用“sudo”命令的，这时我们就需要将登陆的用户加入“/etc/sudoers”文件中。

  • 切换到超级用户，打开“/etc/sudoers”文件：

      $ su -
  $ vi /etc/sudoers

  • 修改文件内容：

    找到下面这行，并在下面插入新行：

      root    ALL=(ALL)       ALL

    内容是：

      xxx    ALL=(ALL)       ALL   # xxx是你的用户名

    这个文件是只读的，保存时需要加“!”，强制保存。

2.2 通过 Software Collection 安装软件

Software Collection 可以方便的管理软件的不同版本，我们通过安装git来说明。

CentOS 7 默认的git版本不是新版，我们通过如下命令安装新版软件：

$ sudo yum install centos-release-scl # 安装 Software Collection
$ sudo yum install rh-git29 # 安装新版 git
$ scl enable rh-git29 bash # 通过 scl 启用新版软件
$ git --version
git version 2.9.3

记住，如果不使用第三个命令，git版本依旧是旧版。

更多软件的安装可以参考官网：Software Collection
