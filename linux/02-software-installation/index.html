2.1 添加用户到“sudoers”

在装完系统后，CentOS 默认普通用户是无法使用“sudo”命令的，这时我们就需要将登陆的用户加入“/etc/sudoers”文件中。

  • 切换到超级用户，打开“/etc/sudoers”文件：

      $ su -
  $ vi /etc/sudoers

  • 修改文件内容：

    找到下面这行，并在下面插入新行：

      root    ALL=(ALL)       ALL

    内容是：

      xxx    ALL=(ALL)       ALL   # xxx是你的用户名

    这个文件是只读的，保存时需要加“!”，强制保存。

2.2 通过 Software Collection 安装管理软件包

Software Collection 可以方便的管理软件的不同版本，我们通过安装git来说明。

2.2.1 安装 Software Collection

$ sudo yum install centos-release-scl

2.2.2 搜索软件包

$ yum search git

2.2.3 安装软件包

$ sudo yum install rh-git29

软件包安装后，可以查看安装情况，该命令会列出安装过的所有软件包：

$ scl --list

2.2.4 启用软件包

$ scl enable rh-git29 bash

注意，这些软件包在重启后不会激活。SCL 的设计初衷就是在不影响原有配置的前提下，让新旧软件能一起运行。不过你可以通过~/.bashrc加载 SCL 提供的 enable 脚本来实现自动启用。

source /opt/rh/rh-git29/root/usr/bin/git/enable

更多软件的安装可以参考官网：Software Collection
