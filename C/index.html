C语言学习笔记

C语言和系统编程紧密相关，学好C语言可以从系统编程着手。参考博文《如何学好C语言》

  • 语言：学好C语言的两大主题，指针和内存管理
    • 《C语言程序设计：现代方法》是一本涵盖C89&C99大而全的一本入门教材。作者很用心地编排各个主题，以及相关知识点的贴心标注，使得本书非常适合入门者学习。
    • 《C语言的科学和艺术》集中讨论库和抽象的用法。初学者很容易陷入C语言细节，本书着眼于程序设计，强调软件工程，恰恰规避了这些细节。
    • 《C和指针》、《C专家编程》、《C陷阱与缺陷》这三本书涵盖了C语言的所有细节，可作为手边工具书查阅。
  • 算法和数据结构
    • 《C程序设计的抽象思维》是《C语言的科学和艺术》的姊妹篇，从软件工程技巧出发，介绍了数据结构和算法的基础知识，学习程序设计不可多得的好书。
  • 系统编程
  • 系统设计

对于初学者，往往会遇到一些瓶颈期。如果你也遇到这样的遭遇，不妨先把手头的书放一放，去图书馆或网络找一些其他的书看一看。不同作者会有不同的描述方式和侧重点，比如《Head First C》、《Learn C the Hard Way》。

这些思维方式的碰撞，可能有助于你度过瓶颈期。但切记，当你决定看一本书，一定要完整的读完，不要放过任何一行知识点。毕竟作者们都是一个字一个字写出来的，我们读者又有什么理由不尊重一下呢。

共勉！
